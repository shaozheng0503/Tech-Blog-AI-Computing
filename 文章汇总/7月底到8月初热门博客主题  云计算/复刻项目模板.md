# Serverless AI 项目复刻模板

## 🎯 快速开始

### 项目结构
```
serverless-ai-template/
├── backend/          # FastAPI后端
├── frontend/         # React前端
├── infrastructure/   # 部署配置
└── docs/            # 文档
```

### 核心技术栈
- **后端**: FastAPI + Docker + Serverless
- **前端**: React + Vite + Ant Design
- **部署**: AWS Lambda / 阿里云函数计算

### 快速部署
```bash
# 1. 克隆模板
git clone https://github.com/template/serverless-ai-template

# 2. 配置环境
cp .env.example .env

# 3. 部署
serverless deploy
```
